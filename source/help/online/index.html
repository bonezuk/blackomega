Overview

In the beginning before the Big Bang Kilonova Hypothesis there was Black Omega. Perfected over 25 years, Black Omega is an open-source music player designed for audiophile renderings of your audio files with bit perfect accuracy. Employing low level audio APIs and custom codecs Black Omega utilises the capabilities of your premium high-end DAC delivering pure seamless playback of your music collection.

Firstly you'll need to add music files to create your playlist. This can be done via

  1. Add files and folders using a file selection dialog.
  2. Drag files and folders into your playlist from Windows Explorer or MacOS Finder.
  3. Copy and Paste from Windows Explorer or Mac OSFinder.
  4. Through file association. (Windows)

Then you control playback via the Playback Controls.

Open the Preferences dialog to setup Audio, Controls and File Association.

Black Omega provides codec support for the following audio file formats.

Format File Extension(s) Codec
Waveform Audio Format.wav, .waveBlue Omega
MPEG-1/2 Audio Layer III.mp3Black Omega
Vorbis Ogg.oggSilver Omega
MPEG-4 AAC.mp4White Omega
Apple Lossless.mp4, .cafRed Omega
Musepack.mpc, .mp+, .mppCyan Omega
FLAC, Free Lossless Audio Codec.flacGreen Omega
AIFF, Audio Interchange File Format.aiff, .aif, aifcViolet Omega
Wavpack.wvWavpack Omega

Playlists can be loaded and saved using the m3u, pls and xspf playlist file formats.
